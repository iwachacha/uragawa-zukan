# Motif Suitability Gate

世界観に合う題材でも、AI画像生成やグッズ化に向かない場合は主役にしない。

## 主役に向く題材

- 一目でわかる外形がある。
- 正方形または縦長の構図に収まる。
- 3cm程度のステッカーでも認識できる。
- 現実スケールを多少崩しても、元モチーフとして納得できる。
- 表面、影、穴、断面、反射、持ち方など、裏事情を大きく見せる余地がある。
- 色や形にアクセントを作れる。

## 主役に向きにくい題材

- 小さすぎる粒、粉、カス、汚れ、点。
- 白背景で埋もれやすい白い小物。
- スケールを大きくすると別物に見えるもの。
- 裏事情が細部や文字情報に依存するもの。
- ただの線、紐、点に誤認されやすいもの。
- 商品化時に横長以外の展開が難しいもの。

## 扱いを変える

主役に弱い題材は、次の扱いへ回す。

- 強い主役モチーフの周辺現象にする。
- SNS用の小ネタにする。
- 背景の痕跡や関係性として使う。
- 総柄の小要素にする。
- 台帳登録せず、`continuity_notes.md` にだけ残す。

## 事前チェック

制作前に以下を確認する。

- 商品化フォーマットは `square`, `vertical`, `one-point`, `pattern`, `sticker` のどれに向くか。
- 横長専用になっていないか。
- 主役と周辺物のスケール比が現実から離れすぎないか。
- AIが別物として解釈しやすい要素はないか。
- 裏事情を大きな形や行動で見せられるか。

## 消しカスからの学び

消しカスは世界観の題材としては使えるが、主役商品には弱い。

- 小さく白く、背景に埋もれやすい。
- 大きくすると消しカスに見えない。
- 黒い鉛筆線が紐に誤認されやすい。
- 魅力を足すほど別キャラクター化する。

今後は、消しカスを主役にせず、消しゴムやノートの周辺現象として扱う。
