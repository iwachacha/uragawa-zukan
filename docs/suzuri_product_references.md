### 商品別の設計メモ

#### ステッカー

- 630 x 630 px、8 x 8 cm。
- 透過PNGの形に沿ってカットされるため、外形が重要。
- 3cm程度に縮小しても主役と裏事情が読める構成にする。
- 細い線や小さな文字に世界観を置かない。

#### Tシャツ / パーカー

- 中央配置は 2520 x 2992 px 系が多い。
- 遠目で見てもわかるシルエット、余白、色数が重要。
- 主張が強すぎないワンポイント版も作れる題材が強い。
- 春夏はTシャツのラインナップを充実させるとよい、という公式販売Tipsがある。

#### トートバッグ

- Tote Bag は 2362 x 2362 px の正方形。
- 白・生成り系の布に映える線と色を意識する。
- 中央に置いたとき、絵として弱くならない主役外形が必要。

#### スマホケース

- iPhone系は縦長が多く、多くの機種で 1616 x 2721 px。
- 縦長切り出しでも主役が弱くならない構図にする。
- Character Icon は中央または下部、Scene Emblem は縦の流れを作る。

#### アクリルキーホルダー / アクリルスタンド

- Acrylic Key Chain は 1102 x 1102 px。
- Acrylic Stand は本体 3688 x 3688 px、台座 2033 x 2033 px。
- アクリルスタンドでは、1mm以下のオブジェクト、3px以下の線や文字、不透明度10%未満の要素は再現や加工に問題が出る可能性がある。
- 極端に細い形状や、パーツ同士が離れている形は破損や販売保留のリスクがある。

## ウラガワ図鑑への反映

### 優先する画像戦略

1. `Character Icon`: ステッカー、Tシャツ、トート、スマホケースへ展開しやすい。
2. `World Object`: 主役の体内や殻、葉、花、断面に小さな世界を入れる。背景なしでも世界観が伝わる。
3. `Relationship Pair`: ペア商品、ステッカーセット、SNSシリーズに向く。単体化案も同時に考える。
4. `Scene Emblem`: 世界観は強いが、商品展開はアクリルブロック、ポスター、スマホケース寄り。
5. `Pattern Seed`: 総柄、マステ、小物向き。単体アイコンとしても成立する要素だけを使う。

### 商品化に強い特徴

- 背景なしでも成立する強い外形。
- 3cmに縮小しても伝わる主役と行動。
- 白・生成り・黒など複数の商品色で埋もれにくい線と配色。
- 正方形と縦長の両方に再構成できる。
- 細部ではなく、大きな形、動き、関係性、体内世界で裏事情が伝わる。
- 女性中心ターゲットが持ち物に貼る、着る、持つことへ抵抗が少ないモチーフ。

### 避ける設計

- 横長専用の構図。
- 小さな文字、細線、微細な断面だけで世界観を説明する。
- 白い小物を白背景に置くだけ。
- 商品化時に主役が小さくなり、何が起きているかわからない構図。
- アクリル化したときに折れそうな細い突起や離れたパーツ。
