# Visual Strength Gate

世界観として成立していても、見た目が弱い案は通さない。SNSや商品化では、説明前の第一印象が弱いものは基本的に差し戻す。

## 必須の強さ

各観測案は、画像生成前に以下のうち最低2つを明確に持つ。

- **キャラクター性**: 顔でなくても、姿勢、動き、群れ方、癖で存在として覚えられる。
- **ストーリー性**: 一枚絵の中で何かが起きている。運ぶ、隠す、逃げる、待つ、守る、奪う、片付けるなど。
- **シルエットの強さ**: 遠目でも外形が面白い。ステッカーの形として欲しくなる。
- **シュールさ**: 普通の物体が妙な仕事をしている、または人間が知らない営みをしている。
- **かわいさ・怖かわいさ**: 所有したくなる感情がある。ただし甘い顔を付けるだけにしない。
- **グラフィック性**: 線、模様、配置、余白、反復がデザインとして美しい。

## 差し戻し条件

以下に当てはまる場合は、世界観テキストが良くても差し戻す。

- ただ物が置かれているだけに見える。
- 裏事情が小さすぎて一瞬で伝わらない。
- 視覚的な主役が弱い。
- かわいさ、シュールさ、怖さ、おしゃれさのどれも弱い。
- 商品にしたとき「なぜ欲しいか」が言えない。
- 説明文を読まないと魅力が発生しない。

## 画像生成前の問い

- 3cmのステッカーになっても何が起きているかわかるか。
- サムネイルで見たとき、普通のイラストと違う点が見えるか。
- この存在をグッズにしたとき、人は何に惹かれるのか。
- 顔をなくしても魅力が残るか。顔を付けるなら、顔以外の魅力もあるか。
- 動き、群れ、道具、痕跡、対比のどれかで画面にストーリーがあるか。

## プロンプト必須項目

画像生成プロンプトには、通常項目に加えて以下を入れる。

- 視覚的な主役
- 画面内で起きている行動
- ステッカーにしたときの外形
- 遠目で見えるアクセント
- 避ける弱い表現

## 弱い案の救済方法

- 物体を少し生き物のように動かす。ただし顔だけに頼らない。
- 群れや行列にして、習性を見せる。
- 大小差を付け、主役を作る。
- 裏事情を断面の小さい情報ではなく、外形や動作に出す。
- 置き絵ではなく、何かの最中の一枚にする。
- 商品化時は、背景説明を削ってシルエットと行動に集中する。
