# Visual Strategy

画像化前に、何を主役にするかを決める。存在単体を主役にするのか、世界ごと主役にするのかで設計を変える。

## 戦略タイプ

### 1. Character Icon

存在そのものを主役にする。SUZURI向きの基本形。

- 背景なし、または極薄背景
- 正方形または縦長
- シルエットが強い
- ステッカー、Tシャツ、スマホケースに展開しやすい
- 例: カタツムリの殻に雨雲、ダンゴムシの月甲冑

### 2. World Object

存在の体や断面に小さな世界を入れる。

- 背景を作り込まず、主役の中に世界を宿す
- 商品化しやすく、世界観も伝わりやすい
- 例: 種の中の予備の庭、クラゲの記憶灯、松ぼっくりの鍵束

### 3. Relationship Pair

2つの存在の関係性を主役にする。

- ペア構図
- 視線、糸、光、根、影、雨などで関係を見せる
- ステッカーセットやペア商品に向く
- 例: きのこと傘の雨盟約、影を干す猫と朝顔

### 4. Scene Emblem

背景込みで、小さな世界の一場面を見せる。

- 世界観は強いが、商品化は難しくなりやすい
- 背景は丸い窓、切り株、葉、月、瓶などのフレーム内に収める
- ポスター、アクリルブロック、スマホケース向き
- 例: 鳥居の下の風の巣、金魚鉢の夜空係

### 5. Pattern Seed

総柄や小物向けに、小さな要素の反復を作る。

- 1つ1つの情報量は少なくても、並ぶと魅力が出る
- 例: どんぐりの帽子職人、夜露の羊

## 商品化前に決めること

- メインビジュアルは何か。
- 背景なしで成立するか、背景込みで世界ごと売るか。
- どの商品を主戦場にするか。
- どの層に刺すか。
- どこまで描き込むか。
- 3cmのステッカーで残す情報は何か。
- スマホケース縦長にしたとき、主役が弱くならないか。
- SUZURIの推奨サイズと人気アイテムに合うか。詳しくは `docs/suzuri_product_references.md` を見る。

## 基本優先順位

1. 商品化しやすい `Character Icon` または `World Object` をまず考える。
2. 世界観が強い案は `Relationship Pair` や `Scene Emblem` も検討する。
3. 背景込みにする場合も、主役の外形を弱くしない。
4. 情報量は「見える世界が強い」方向に使い、ただの背景装飾に使わない。

## SUZURI向けの主戦場

- まず `Sticker`, `T-Shirt`, `Tote Bag`, `Smartphone Case` を想定する。
- 形が強い存在は `Acrylic Key Chain` も検討する。
- 世界ごと見せる案は `Acrylic Block`, `Stickable Poster`, `Smartphone Case` も検討する。
- 横長ステッカー専用の案は、SNS反応が強い場合のみ商品化候補にする。
