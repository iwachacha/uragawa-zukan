# Visual Language

ウラガワ図鑑の絵は、弱い現実再現でも、ただのマスコットでもなく、身近な存在の裏側が一瞬見えたような商品向けイラストを目指す。

## 基本の見た目

- 手描き感のある少し太めの主線。
- 主線は黒ベタではなく、濃い茶、濃いグレー、深い青緑など少しやわらかい色を優先する。
- 色数は少なめ。主役色 1-2 色、アクセント 1 色、影 1 色を基本にする。
- 余白は多め。ただし余白で弱く見える場合は、主役の外形、行動、アクセントを強める。
- 背景は白、透明、淡い一色、または小さな地面・影程度を基本にする。

## 主役の強さ

商品化候補の主役は次のうち最低2つを持つ。

- 強いシルエット
- 画面内の行動
- 表情または姿勢
- 体内世界、影、反射、断面などの裏事情
- 遠目で見えるアクセント色
- ペアや小さな群れの関係性
- 所有したくなる外形

## 情報量

- ぱっと見は単純、近づくと裏事情がわかる状態にする。
- 細部だけに世界観を置かない。
- 小さな文字、薄すぎる線、白いものを白背景に置くだけの構図は避ける。
- 画面内で「何かが起きている」ことを優先する。

## かわいさの扱い

かわいさは入口として使うが、着地点にしない。

- かわいい + 奇妙な行動
- かわいい + 少し不穏な習性
- かわいい + 世界が広がる体内構造
- かわいい + 妙に納得できる労働感
- かわいい + おしゃれな形や配色

## 推奨ビジュアル構造

### Character Icon

背景なしで主役を見せる。ステッカー、Tシャツ、トートに強い。  
主役の行動か裏事情を外形に入れる。

### World Object

主役の体内、殻、葉、花、断面、影に小さな世界を入れる。  
単体でも世界観が伝わりやすい。

### Relationship Pair

2つ以上の存在の関係性を見せる。  
ペアステッカーやシリーズ展開向き。単体化できる主役を必ず残す。

### Scene Emblem

背景込みで小さな世界を見せる。  
スマホケース、アクリルブロック、ポスター向き。Tシャツでは情報過多にならないよう簡略版を作る。

### Pattern Seed

総柄や小物に展開するための単位絵。  
1つだけ抜き出しても成立する形にする。

## 避ける見た目

- 横長専用で、主役が小さい。
- 背景が広く、主役の魅力が薄い。
- 現実の物体を少し擬人化しただけ。
- 表情だけで裏事情を説明している。
- 文字やキャプションがないと成立しない。
- 不快な虫らしさ、リアルな脚、群がり、複眼、卵、寄生感が前面に出る。
