# Structured Image Prompt

gpt-image2.0向けプロンプトは、長い説明文ではなく、優先順位が伝わる構造で書く。

## 推奨形式

```text
[GOAL]
商品化向けのステッカー風イラスト。最重要は、{誰に刺す魅力} と {一目で見える行動}。

[SUBJECT]
主役：
脇役：
モチーフ認識に必要な現実要素：

[ACTION]
画面内で起きていること：
役割差：
ストーリーの瞬間：

[HIDDEN SIDE]
見えていない裏事情：
絵で見せる方法：

[COMPOSITION]
primary format:
secondary use:
target product:
master canvas:
構図：
主役と周辺物の相対スケール：
シルエット：
余白：
商品配置：

[STYLE]
線：
色：
背景：
質感：

[ACCENT]
遠目で見えるアクセント：
刺さる魅力を強める要素：

[AVOID]
避ける表現：
弱くなる表現：
文字・説明：
```

## 書き方

- 1セクションに1-3文まで。
- 抽象語だけで終わらせない。
- 「かわいい」「不思議」より、何がかわいいか、何が不思議かを書く。
- AIにとって重要な優先順位を上から順に置く。
- 禁止事項は最後にまとめる。
- 商品化用は、背景より主役の外形と行動を優先する。
- 原則として `primary format: square` を指定する。横長専用にしない。
- 主役と周辺物の相対スケールを書く。
- `target product` と `master canvas` を書く。例: `Sticker/T-Shirt/Tote`, `3000 x 3000 transparent PNG`.

## 悪い例

```text
消しゴムのカスが線を運んでいる、かわいくて不思議な雰囲気のイラスト。
```

## 良い例

```text
[GOAL]
商品化向けの横長ステッカー。小さな白い消しカス部隊が、消された黒い線を夜勤のように回収している健気さを最重要にする。

[SUBJECT]
主役は先頭の大きな消しカス隊長。丸い白い体、薄いグレーの輪郭、短い粉のような脚。脇役は小さな消しカス3-5匹。

[ACTION]
部隊が黒い鉛筆線をロープのように引っぱる。1匹は見張り、1匹は線を巻き取り、1匹は落ちた文字片を拾う。
```
